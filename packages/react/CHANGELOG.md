# effect-mermaid-react

## 0.1.0

### Minor Changes

- Initial release of effect-mermaid packages:

  - @effect-mermaid: Core types, interfaces, and stub implementation
  - @effect-mermaid-node: Node.js implementation with real Mermaid.js rendering
  - @effect-mermaid-react: React components for browser-based diagram rendering

  Features:

  - Full TypeScript support with Effect framework
  - Multiple rendering environments (Node.js, React)
  - Comprehensive error handling and type safety
  - Theme and configuration support
  - Testing utilities included

### Patch Changes

- Updated dependencies
  - effect-mermaid@0.1.0
